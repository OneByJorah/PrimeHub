## Why This Exists

JorahOne is the organization behind 29 open-source projects focused on self-hosted infrastructure. This meta-repo catalogs the entire portfolio — NOC dashboards, AI agents, monitoring tools, deployment stacks, and MSP utilities — all standardized with MIT licensing, security policies, and consistent brand conventions.
